# Portfolio Website

Single-page portfolio inspired by the clean, project-forward style of seanwirkus.com, customized for Rahul Sengupta.

## Files

- `index.html` — page structure and sections
- `styles.css` — theme, layout, responsive styles
- `script.js` — all content data and rendering

## Customize with your resume

Update the `portfolioData` object in `script.js`:

- `hero.headline` and `hero.subtext`
- `about`
- `skills`
- `projects` (title, description, tags, link)
- `experience` (role, company, range, bullets)
- `contact.links` (email, LinkedIn, GitHub)

## Run locally

Open `index.html` directly in your browser, or run a local static server from this folder.
